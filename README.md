# munger-skill

基于查理·芒格 11 篇演讲蒸馏的投资与思维体系 skill。

## 与 buffett-skill 的根本差异

| 维度 | buffett-skill | munger-skill |
|------|-------------|-------------|
| 分析起点 | 企业基本面 | 人类心理与认知偏误 |
| 核心工具 | 护城河 + 财务分析 | 多元思维模型格子 |
| 最大原创贡献 | 伟大企业定价框架 | 系统化人类误判心理学 |
| 问题解法 | 正向分析 | 逆向思维优先 |
| 独特模块 | 行业手册、所有者收益 | Lollapalooza效应、学术批判、道德系统设计 |

## 三种分析路径

```
A · 快速筛选   → 10问检查表（无需读参考文件）

C · 专题问题   → 直接跳对应文件
```

## 素材来源

基于以下 11 篇芒格演讲提炼：

1. 哈佛大学毕业典礼演讲（1986）— 逆向思维的经典示范
2. 初级世俗智慧（1994）— 多元思维模型格子核心阐述
3. 世俗智慧与投资（1995）— 投资应用与心理学整合
4. 关于实用思维的实用思考（1996）— 可口可乐案例 + Lollapalooza
5. 多学科技能的必要性（1998）— 对单一学科化的批判
6. 领先慈善基金会的投资实践（1998）— 集中投资理论
7. 慈善圆桌早餐演讲（2000）— Febezzlement 概念
8. 2003 年金融大丑闻（2003）— 股票期权道德批判
9. 学术经济学的优缺点（2003）— 系统性经济学批判
10. USC 法学院毕业典礼演讲（2004）— 道德与品格体系
11. 人类误判心理学（2005）— 25+ 心理偏误完整体系
